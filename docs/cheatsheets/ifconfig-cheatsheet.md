# ifconfig: Cheatsheet

> Configure a network interface.

```bash
# Display all interfaces
ifconfig

# Display all interfaces, even if down
ifconfig -a

# Disable an interface
ifconfig <𝗜𝗡𝗧𝗘𝗥𝗙𝗔𝗖𝗘> down

# Enable an interface
ifconfig <𝗜𝗡𝗧𝗘𝗥𝗙𝗔𝗖𝗘> up
```
