# type: Cheatsheet

> Display the type of command the shell will execute.

```bash
# Display the type of a command
type <𝗖𝗢𝗠𝗠𝗔𝗡𝗗>

# Display all locations containing the specified executable
type -a <𝗖𝗢𝗠𝗠𝗔𝗡𝗗>

# Display the name of the disk file that would be executed
type -p <𝗖𝗢𝗠𝗠𝗔𝗡𝗗>
```
