# hostname

```bash
# Show current hostname
hostname

# Show hostname's network address
hostname -i

# Show all host's network addresses
hostname -I

# Show the Fully Qualified Domain Name (FQDN)
hostname --fqdn

# Set current host name
hostname new_hostname
```
