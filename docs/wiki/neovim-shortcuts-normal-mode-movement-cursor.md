# Neovim Shortcuts: Normal Mode: Movement: Cursor

| Key          | Description        |
| ------------ | ------------------ |
| `h` or `[←]` | Move cursor left.  |
| `j` or `[↓]` | move cursor down.  |
| `k` or `[↑]` | move cursor up.    |
| `l` or `[→]` | move cursor right. |
