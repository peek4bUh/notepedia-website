# Dot

Matches any character except a line break. It will match any character in that character position.

```
[tux@linuxbox ~]$ ls /bin | grep '.zip'
bzip2
bzip2recover
gzip
bunzip2
gunzip
funzip
gpg-zip
prezip
unzip
```
