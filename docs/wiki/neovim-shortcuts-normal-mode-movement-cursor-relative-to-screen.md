# Neovim Shortcuts: Normal Mode: Movement: Cursor Relative to Screen

| Key  | Description               |
| ---- | ------------------------- |
| `H`  | Move to top of screen.    |
| `M`  | Move to middle of screen. |
| `L`  | Move to bottom of screen. |
| `zz` | Center cursor on screen.  |
