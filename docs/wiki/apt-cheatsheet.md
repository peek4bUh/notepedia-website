# apt: Cheatsheet

```bash
# List all installed packages
apt list --installed

# Install a package
apt install <𝗣𝗞𝗚_𝗡𝗔𝗠𝗘>

# Install a package without confirmation prompt
apt install <𝗣𝗞𝗚_𝗡𝗔𝗠𝗘> -y
```
